# 🛒 E-commerce Testing Project

## 📌 Project Overview

This project demonstrates end-to-end manual testing of an e-commerce web application. It includes test planning, test case design, test execution, bug reporting, and test summary preparation.

---

## 🌐 Application Tested

Testing was performed on the SauceDemo website, a demo e-commerce application widely used for practicing manual testing.

---

## 🧪 Modules Tested

* Login
* Product
* Cart
* Checkout
* Logout

---

## 🔍 Testing Types Covered

* Functional Testing
* UI Testing
* Negative Testing
* Data Validation Testing

---

## 📂 Project Structure

* Test Plan
* Test Cases
* Bug Reports
* RTM (Requirement Traceability Matrix)
* Test Summary Report
* Screenshots

---

## 🐞 Key Defect Identified

* Product image is not displayed in Cart and Checkout pages
